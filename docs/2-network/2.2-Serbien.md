## Topologie

<p align="center">
  <img
    src="../../assets/topology_rs.png"
    alt="Netzwerk-Topologie Standort Serbien"
    width="100%"
  >
</p>

Der serbische Standort beherbergt den zentralen Proxmox-Core-Node sowie die WLAN-Infrastruktur. Alle Geräte hängen hinter einer dedizierten OPNsense-Firewall. Die Access Points werden direkt über den PoE-Switch versorgt.

<hr style="border: none; height: 2px; background: linear-gradient(to right, transparent, #555, transparent); margin: 30px 0;">

## Logischer Datenfluss

```text
[ Internet / ISP ]
        |
 [ Huawei Router ]      ← Modem / ISP-Einspeisung
        |
  [ OPNsense ]          ← Firewall · NAT · VPN-Endpunkt
        |
 [ PoE Switch ]         ← Zentraler Verteiler (PoE-fähig)
    /   |   \
[Node] [AP1] [AP2]      ← Proxmox Core Node & Access Points
```
