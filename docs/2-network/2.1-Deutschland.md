## Topologie

<p align="center">
  <img
    src="../../assets/topology_de.png"
    alt="Netzwerk-Topologie Standort Deutschland"
    width="100%"
  >
</p>

Der deutsche Standort bildet einen vollständigen Netzwerkknoten mit eigener Firewall, kaskadierten Switches und mehreren Proxmox-Nodes. Die Firewall übernimmt Routing, NAT und VPN-Terminierung – der ISP-Router arbeitet ausschließlich als Modem.

<hr style="border: none; height: 2px; background: linear-gradient(to right, transparent, #555, transparent); margin: 30px 0;">

## Logischer Datenfluss

```text
[ Internet / ISP ]
        |
  [ FritzBox ]          ← Modem / DSL-Einspeisung
        |
  [ OPNsense ]          ← Firewall · NAT · VPN-Endpunkt
        |
  [ Switch 1 ]          ← Core-Verteiler (unmanaged, kein PoE)
        |
  [ Switch 2 ]          ← Arbeitsplatz-Switch (kaskadiert, kein PoE)
     /      \
[Node 1]  [Node 2]      ← Proxmox Hypervisoren
```

<hr style="border: none; height: 2px; background: linear-gradient(to right, transparent, #555, transparent); margin: 30px 0;">
