| Standort       | Gerät            | Rolle                                                                       |

| 🇩🇪 Deutschland | FritzBox         | DSL-Einspeisung · Modem-Betrieb · WAN-Übergabe an OPNsense                  |

| 🇷🇸 Serbien     | Huawei Router    | ISP-Einspeisung · Modem-Betrieb · WAN-Übergabe an OPNsense                  |
| 🇷🇸 Serbien     | PoE Switch       | Zentraler Rack-Switch · verbindet OPNsense & Proxmox · versorgt APs per PoE |
| 🇷🇸 Serbien     | TP-Link AP 1     | Indoor Access Point · Deckenmontage · primäre WLAN-Abdeckung                |
| 🇷🇸 Serbien     | TP-Link AP 2     | Outdoor Access Point · erweiterte Reichweite für externe Client-Zonen       |
